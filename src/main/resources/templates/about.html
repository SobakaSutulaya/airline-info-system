Опыт работы с технологиями

В процессе создания проекта были использованы современные технологии веб-разработки и Java-экосистемы. Приобретён опыт в построении многоуровневой архитектуры, работе с базами данных, реализации безопасности и создании адаптивных пользовательских интерфейсов.

Backend
  • Java — язык разработки
  • Spring Boot — фреймворк
  • Spring Security — безопасность
  • Hibernate JPA — ORM
Frontend
  • HTML — семантическая разметка
  • CSS — адаптивный дизайн
  • JavaScript — интерактивность
  • Thymeleaf — шаблонизатор
Инструменты
  • MySQL — база данных
  • Maven — управление зависимостями
  • Git — контроль версий
  • REST API — архитектура
